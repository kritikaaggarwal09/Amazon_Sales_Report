# Amazon Sales Dashboard – Power BI Project

## Overview
This project present an interactive Power BI dashboard built using Amazon Sales Data. The dashboard is designed to provide meaningful business insights through data visualization and analytics. It helps users understand sales performance, customer behavior, product trends, and overall business growth.

The report contains two interactive tabs, each focused on different aspects of the sales analysis.

---

## Dashboard Features

### 1. Sales Overview Dashboard
This page provides overview of Amazon sales insights including:

- Total Sales
- Year-To-Date(YTD) Sales
- Quater-To-Date(QTD) Sales
- YTD Product Sold
- Yearly Sales Analysis
- Monthly Sales Analysis
- Weekly Sales Analysis
- Product Wise Sales
- Top 5 Products by YTD Sales
- Average Product Price
- Total Sales by each Product

### 2. Product Analysis Dashboard
This tab focuses on product performance and detailed sales insights using interactive visuals and filters.
